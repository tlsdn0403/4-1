11

14

15

낙천적 동기화는 충돌이 드물다고 가정하고 일단 진행한 뒤 validation을 수행하는 방식이다. ( T / F )

volatile int victim = 0;
volatile bool flag[2] = { false, false };

void Lock(int myID)
{
    int yourID = 1 - myID;
    flag[myID] = true;
    victim = myID;
    while (flag[yourID] && victim == myID) { }
}
