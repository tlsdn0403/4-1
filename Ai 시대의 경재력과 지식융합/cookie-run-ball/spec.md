# Rolling Dash 프로젝트 명세서

## 1. 프로젝트 개요
- 이름: Rolling Dash
- 목표: 쿠키런처럼 자동으로 전진하는 주인공이 장애물을 피하고 코인을 모으며 오래 살아남는 2D 점프 게임
- 최종 형태: 로컬 브라우저에서 실행되는 HTML/CSS/JavaScript Canvas 게임
- 외부 이미지: 사용하지 않음. 공, 가시, 벽, 절벽, 배경을 모두 Canvas 도형으로 표현

## 2. 기술 스택
- 언어: HTML, CSS, JavaScript
- 렌더링: Canvas API
- 실행 환경: Windows 브라우저
- 외부 라이브러리: 없음
- 저장 방식: 브라우저 localStorage에 최고 점수 저장

## 3. 핵심 기능
- 플레이어는 동그란 공으로 표현한다.
- 게임이 시작되면 화면이 자동으로 오른쪽 진행 느낌을 낸다.
- Space 또는 위쪽 화살표로 점프한다.
- 짧게 누르면 낮게, 길게 누르면 조금 더 높게 점프한다.
- 공중에서 한 번 더 점프할 수 있는 이단점프를 지원한다.
- 장애물은 가시, 벽, 높은 벽, 절벽으로 구성한다.
- 코인을 먹으면 점수가 오른다.
- 공중 플랫폼을 밟아 점프 리듬을 이어갈 수 있다.
- 일부 넓은 절벽과 높은 벽은 플랫폼을 활용해야 통과하기 쉽다.
- 장애물에 닿거나 절벽에 떨어지면 게임이 종료된다.
- P 키로 일시정지, R 키로 재시작한다.

## 4. 사용자 흐름
1. 사용자가 `index.html`을 연다.
2. 시작 화면에서 버튼 또는 Space 키를 누른다.
3. 공이 자동 진행하며 장애물이 등장한다.
4. 사용자는 점프와 이단점프 타이밍을 맞춰 장애물을 피하고 코인을 먹는다.
5. 충돌 또는 추락 시 기록 종료 화면이 뜬다.
6. 다시 시작 버튼 또는 R 키로 재도전한다.

## 5. 평가 기준
- 조작이 즉시 반응하는가
- 이단점프가 착지 전 한 번만 가능한가
- 점프 높이가 지나치게 낮거나 높지 않은가
- 가시, 벽, 높은 벽, 절벽, 플랫폼이 서로 다른 플레이 감각을 만드는가
- 코인이 점수 기준으로 작동하는가
- 충돌 판정이 납득 가능한가
- 점수와 최고 기록이 정상 표시되는가
- 외부 이미지 없이도 게임 목적이 이해되는가

## 6. Non-Goals
- 로그인, 서버 저장, 온라인 랭킹은 구현하지 않는다.
- 외부 캐릭터 이미지나 저작권이 있는 소재는 사용하지 않는다.
- 모바일 터치 조작은 이번 버전의 필수 목표가 아니다.
