# Rolling Dash 테스트 체크리스트

- [ ] `index.html`을 열면 시작 화면이 보인다.
- [ ] 시작 버튼을 누르면 게임이 시작된다.
- [ ] Space 또는 위쪽 화살표를 누르면 공이 점프한다.
- [ ] 공중에서 한 번 더 누르면 이단점프가 된다.
- [ ] 착지 전 세 번째 점프는 되지 않는다.
- [ ] 착지하면 이단점프 횟수가 다시 회복된다.
- [ ] 키를 길게 누르면 짧게 누를 때보다 약간 더 높게 점프한다.
- [ ] 코인을 먹으면 코인 점수가 오른다.
- [ ] 가시에 닿으면 게임 오버가 된다.
- [ ] 벽에 닿으면 게임 오버가 된다.
- [ ] 높은 벽은 플랫폼이나 이단점프를 활용해야 통과하기 어렵다.
- [ ] 절벽에 빠지면 게임 오버가 된다.
- [ ] 공중 플랫폼 위에 착지할 수 있다.
- [ ] 넓은 절벽 위 플랫폼을 밟으면 통과할 수 있다.
- [ ] 거리가 증가할수록 속도가 조금씩 빨라진다.
- [ ] P 키로 일시정지와 계속하기가 가능하다.
- [ ] R 키로 재시작할 수 있다.
- [ ] 최고 점수가 새로고침 후에도 유지된다.
- [ ] 외부 이미지 파일 없이 실행된다.
